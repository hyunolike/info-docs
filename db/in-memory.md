## 인메모리 데이터베이스
> [참고자료](https://velog.io/@rudwnd33/TIL-10%EC%9B%94-27%EC%9D%BC)
- 디스크가 아닌 주 메모리에 모든 데이터를 보유하고 있는 데이터베이스
  - 디스크 검색보다 자료 접근이 훨씬 빠른 것이 장점
- 인메모리 방식: 메모리상에 색인(인덱스) 넣어 필요한 모든 정보를 메모리상의 색인을 통해 빠르게 검색 가능
