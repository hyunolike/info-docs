### 메타태그란
- 브라우저와 검색 엔진을 사용할 수 있도록 웹 문서의 정보 포함
- 웹페이지 요약 > 중요중요!!! ⭐
#### ✔ 메타태그 속성
- `http-equiv = "항목명"`
  - 웹 브라우저 서버 > 명령 내리는 속성
  - name 속성 대신 사용 가능
  - 웹 브라우저 전송됬을 떄만 의미 가짐
  - content 속성 정보 / 값을 위한 `HTTP header`  제공
- `content = "정보 값"`
  - meta 정보의 내용
  - `name` `http-equiv` 연관된 값 줌
- `name = "정보 이름"`
  - 몇 개의 meta 정보의 이름 정함
  - 지정하지 않으면 `http-equiv` 와 같은 기능
