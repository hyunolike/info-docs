## `extends` `implements` 차이
> [참고자료](https://wooono.tistory.com/261)
### 💡 상위 객체(부모)의 특징(메소드, 변수 등) '새롭게 구현' vs '그대로 사용'
#### `extends`
- 부모의 특징을 연장해서 사용

#### `implements`(interface 구현)
- 부모의 특징을 **도구** 로 사용해 새로운 특징 만들어 사용

#### `abstract` 
- `extends` `interface` 혼합
- `extedns` 사용하지만 몇 개는 추상 메소드로 구현됨
