## `oninput` `onchange`
> [참고자료](https://coding-food-court.tistory.com/19)

### `oninput`
- `textarea` 태그 >> 데이터 입력 시 바로바로 데이터 넣어주기?
- 유효성 검사할 때 주로 사용
- `input` `textarea` 태그 사용 가능 ✔

### `onchange`
- input 태그의 초점(focus) 잃는 순간 작동
  - input 태그의 데이터 입력하고 다른 곳 클릭 시 작동
- `input` `textarea` `select` 태그 사용 가능 ✔
