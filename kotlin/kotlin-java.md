## Kotlin vs Java

### 1. Kotlin
- JetBrains에서 만든 프로그램 언어
  - 러시아의 섬 이름ㅋㅋㅋㅋ
- JVM에서 동작
- 2011년 7월 최초 공개
- 2017년 5월 Android 공식 언어 채택
- 멀티 패러다임 프로그래밍 언어 ✔ 코프링ㅋㅋㅋㅋㅋ
  -  자바 >> 객체지향 특성, 함수형 언어
  -  파이썬, 자바스크립트 >> 스크립트 파일만으로도 실행 가능
