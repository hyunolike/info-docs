### `Recoil`
- 러닝 커브 낮춰주기 위해 쉽고 직관적인 라이브러리 리코일
### `Zustand` 주스탠드
- 독일어 `상태`
- 간력할 플러스 원칙을 바탕 > 작고 빠르게 확장 가능한 상태 관리 라이브러리
- `발행/구독` 모델
### `Jotai` 조타이
- 아토믹 접근을 가지고 만든 리엑트 상태 관리 도구
- 조타이 기본적인 단위 > 아톰
### `Valtio``
- 프록시 기반의 리엑트 상태 관리 라이브러리
- 통신을 중계하는 대리인 역할
