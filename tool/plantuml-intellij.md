## `PlantUML` intellij에서 사용하는 방법
> [참고자료](https://devyonghee.github.io/2022/02/24/plantUML-install/)
### `Graphviz` 설치
- 그래프 시각화해주는 오픈소스 소프트웨어


```shell
brew install libtool
brew link libtool
brew install graphviz
brew link --overwrite graphviz
```
