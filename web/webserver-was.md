### WAS Web Application Server
- 개념
 - DB 조회 / 로직 처리 > 동적 컨텐츠 제공하기 위해 만들어진 웹서버
 - `Web container` `Servlet Container`
 - `Container`: jsp, servlet 실행시킬 수 있는 소프트웨어
- 기능
 - 프로그램 실행 환경 / db 접속 기능 제공
 - 여러개 트랜잭션 관리
 - 비즈니스 로직수행
